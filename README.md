# Projekt LEPSZEJ Ustawy Anty Spekulacyjnej

Celem niniejszego projektu "Lepszej Ustawy Anty Spekulacyjnej" jest stworzenie przejrzystych, skutecznych i sprawiedliwych ram prawnych, które ograniczą negatywne skutki spekulacji, jednocześnie nie hamując zdrowego rozwoju rynku.

Projekt ten powstał z myślą o stworzeniu aktu prawnego realnie odpowiadającego potrzebom polskiego rynku mieszkaniowego, a nie doraźnym potrzebom politycznym.
